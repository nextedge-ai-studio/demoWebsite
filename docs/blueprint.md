# **App Name**: CodePro Hub

## Core Features:

- Introduction Page: Displays a brief introduction of Wang Xiang-Hao as a Full-stack Engineer, his expertise, and background. Include key skills like PHP, MySQL, Javascript, HTML, and Python.
- About Me Page: Detailed information about Wang Xiang-Hao, including his educational background from Tunghai University, 5 years of work experience, and proficiency in both Windows and Linux environments.
- Portfolio Showcase: Dedicated page to showcase various web development projects with descriptions. Use Generative AI to generate descriptions from just images of Wang's past projects.
- Contact Information: A page including ways to contact Wang, with contact form. Optional connection to a calendar booking tool.

## Style Guidelines:

- Primary color: Dark electric blue (#7DF9FF), reminiscent of glowing interfaces and digital displays, will convey a modern, technological feel.
- Background color: A deep, desaturated cyan (#081F29), evoking the ambience of a tech lab, will ground the interface with sophistication.
- Accent color: Analogous cyan (#7DFFCE), brighter and more saturated, will draw attention to key elements and navigation.
- Font: 'Space Grotesk' sans-serif for a computerized, techy look; use for headlines and short amounts of body text, or pair with 'Inter' for longer body text.
- Modern layout with clear sections for introduction, skills, and projects. Use a grid-based system to maintain a clean, structured feel.
- Use sharp, minimalist icons to represent different skills and technologies, creating a sleek, modern aesthetic.
- Subtle transitions and animations on page load and when hovering over elements to add a layer of interactivity and visual interest.
